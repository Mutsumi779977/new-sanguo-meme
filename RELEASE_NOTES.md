# 新三国梗系统 v3.0 发布说明

## 🎭 新三国梗系统 v3.0 正式发布

**用AI的方式，打开2010版《新三国》。**

一个模块化架构的梗工程系统：收集 → 学习 → 输出。

---

## ✨ 核心特性

| 特性 | 说明 |
|:---|:---|
| **双模式输出** | 纯梗模式（三不原则）/ 角色模式（带动作情绪） |
| **联网搜索增强** | 遇到陌生话题自动搜索后匹配梗 |
| **话题映射** | 自动识别 LCK/科技/娱乐/日常等领域话题 |
| **信息充足度检测** | 智能判断是否需要搜索背景知识（方案A+） |
| **显式搜索** | 用户可主动要求搜索特定内容 |
| **可配置匹配** | 权重公式、温度参数均可自定义 |
| **向量匹配** | 语义相似度（可选，默认关闭） |
| **meta 梗融合** | 自动融合称呼类梗 |

---

## 🎮 使用示例

**电竞话题：**
```
输入: "gen2:0T1"
输出: 不可能，绝对不可能！
```

**科技话题：**
```
输入: "GPT5发布了"
输出: 好方略，不过我想稍作修改
```

**日常闲聊：**
```
输入: "我晚上要吃什么呢？"
输出: 是啊，吃什么
```
