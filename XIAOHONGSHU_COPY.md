## 文案一：功能向

🎭 **当你的AI学会了"不可能，绝对不可能！"**

家人们！我的新三国玩梗agent终于发布v3.0正式版了！

✨ **这次有什么新东西？**

🔥 **双模式输出**
- 纯梗模式：干净利落的梗，不废话
- 角色模式：带动作带情绪，沉浸式体验

🔍 **联网搜索增强**
遇到不懂的话题？自动搜索后再匹配梗！
比如输入"gen2:0T1"，直接输出"不可能，绝对不可能！"

🧠 **智能判断（方案A+）**
AI自己会判断：这个话题需要搜索背景知识吗？
不需要就快速回复，需要就自动联网查

🎮 **话题映射**
LCK电竞、科技数码、日常闲聊...自动识别话题领域，匹配合适的情绪和人物

💬 **显式搜索**
你可以直接说"帮我搜一下xxx"，跳过自动判断，直接查资料生成梗

📝 **完整命令列表**
/录入 - 录入新梗
/查询 - 查梗库
/玩梗 - 手动触发
/模式 - 切换纯梗/角色模式
/称呼 - 生成xx爷称呼
/统计 - 看数据

📦 **怎么玩？**
```
pip install pyyaml
python run.py
```
